### 股票买卖问题 
思路：二维DP，第一个维度是时间，第二个维度是状态，包括当期是否持有股票以及曾经卖出股票的次数。
每个节点dp\[i\]\[j\]的value是在j状态下，第i天结束时的最大收益。
初始化第1天持有股票的所有状态dp\[0\]\[j\]为-prices\[0\],结果为最后一天所有不持有股票的状态的最大值。
